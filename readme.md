# cards

My personal tool for keeping notes.

## Yes

- Notes are files
- CLI only
- Just `vim` for writing
- All formats
- Simple
- Fast

## No

- Sharing
- Multi-user
- Databases

---

## Use it

```bash
go build && go install
cards
```

Keys:

- Navigation: vim keys
- Search: `/`
- Rename: `r`
- Copy to clipboard: `c`
- New: `n`
- Archive: `a` (moves to ~/.cards/.archive)
- Present: `p`
- Quit: `q`
