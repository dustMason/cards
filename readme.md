# cards

My personal tool for keeping notes.

## Yes 👍

- ✅ Notes are files
- ✅ CLI only
- ✅ Just `vim` for writing
- ✅ All formats
- ✅ Simple
- ✅ Fast

## No 🙅‍♂️

- ❌ Sharing
- ❌ Multi-user
- ❌ Databases

---

## Use it

```bash
go build && go install
cards
```

##  Shortcuts

| Action                                          | Key           |
|:------------------------------------------------|:--------------|
| Navigation                                      | vim keys `jk` |
| Search                                          | `/`           |
| Rename                                          | `r`           |
| Copy to Clipboard                               | `c`           |
| New                                             | `n`           |
| Archive (move to .archive dir within CARDS_DIR) | `a`           |
| Present                                         | `p`           |
| Quit                                            | `q`           |
